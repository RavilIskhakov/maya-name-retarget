Инструмент для Maya: ретаргет анимации между ригами по совпадающим именам контролов. Ставит соурс в рест-позу, констрейнит одноимённые контролы, запекает результат на таргет
Maya tool to retarget animation between rigs by matching control names — rest-pose the source, constrain matching controls, then bake onto the target.
